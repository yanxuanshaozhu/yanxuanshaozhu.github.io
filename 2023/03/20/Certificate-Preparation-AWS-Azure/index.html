Certificate Preparation(AWS/Azure)

Posted on 2023-03-20 Edited on 2023-03-21 In Computer Science
Symbols count in article: 802 Reading time ≈ 1 mins.

Azure

I recently passed both Exam AZ-900: Microsoft Azure Fundamentals and Exam DP-900: Microsoft Azure Data Fundamentals. I scored 880/1000 on AZ-900 and achieved a perfect 1000/1000 on DP-900. These examinations are not difficult as long as you prepare for them well. In this post, I will share some of the resources I used to prepare for these exams.

  1. Training | Microsoft Learn: This is the official website for preparing for Microsoft Certification exams. I completed all the learning materials relevant to these exams. They also offer free practice assessments for each course.

  2. Examtopics: ExamTopics has a question bank for both AZ-900 and DP-900. Reviewing all the questions there can be very helpful. Be aware that some answers on ExamTopics might be incorrect, so it's wise to check the discussion area for each question to verify answers, find references, or gain a better understanding.

I hope this post is helpful for your exam preparation.
