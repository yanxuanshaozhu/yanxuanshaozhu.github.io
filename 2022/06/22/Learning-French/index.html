孙辉 《简明法语教程》

这里只列出句子用来背诵，其他的都在笔记上。

法语键盘用法

Leçon Un

  1. Qui est-ce? C'est Pascal.
  2. Qù est-il? Il est à Nice.
  3. Qù est-elle? Elle est à Nice.

Leçon Deux

  1. Est-ce que c'est Philippe? (Est-ce Philippe?) Oui, c'est Philippe.
  2. Que fait-il? El est chercheur.
  3. Est-ce que c'est Fanny? Oui, c'est Fanny.
  4. Que fait-elle? Elle est journaliste.
  5. Est-ce qu'il est à Nice? (Est-il à Nice?)
  6. Est-ce quèlle est à Nice? (Est-elle à Nice?)
  7. chercheur = researcher, acteur = actor, actrice = actress, facteur = postman, factrice = postwoman, journaliste = journalist, styliste = stylist

Leçon Trois

  1. Est-ce que Thomas est avocat? Oui, il est avocat.
  2. Où habite-t-il? Il habite à Paris.
  3. Est-ce que Monique est économiste? Oui, elle est économiste.
  4. Où habite-t-elle? Elle habite à Rome.
  5. pilote = pilot, avocat = male lawyer, avocate = female lawyer, moniteur = male monitor, monitrice = female monitor, professeur = teacher, économiste = economist, cinéaste = filmmaker
  6. Où est-ce qu'elle habite? Elle habite à Nice.

Leçon Quatre

  1. Est-ce que c'est votre classe? Oui, c'est notre classe.
  2. Qui est votre professeur? Monsieur Wang Ling est notre professeur.
  3. Qui est votre chef de classe? Wu Feng est notre chef de classe.
  4. chimiste = chemist, secrétaire = secretary, monsieur = sir, votre = your, classe = class, notre = our, chef de classe = class leader

Leçon Cinq

  1. Qu'est-ce que c'est? C'est un stylo.
  2. Qu'est-ce que c'est? C'est une jupe.
  3. Est-ce que c'est le vélo de Charles? Oui, c'est le vélo de Charles.
  4. Est-ce que c'est la revue de Sophie? Oui, c'est la revue de Sophie.
  5. Quelle est sa couleur? Il est rouge.
  6. Quelle est sa couleur? Elle est noire.
  7. un stylo = a pen, un vélo = a bike, un parfum = a perfume, un livre = a book, un drapeau = a flag, un papier = a piece of paper, un verre = a glass, une jupe = a skirt, une guitare = a guitar, une voiture = a car, une revue = a magazine, une veste = a jacket, a serviette = a tower
  8. rouge = red, vert = masculine green, verte = feminine green, noir = masculine black, noire = feminine black, bleu = masculine blue, bleue = feminine blue, jaune = yellow
  9. zéro = 0, un, une = 1, deux = 2, trois = 3, quatre = 4, cinq = 5, six = 6, sept = 7, huit = 8, neuf = 9, dix = 10

Leçon Six

  1. Est-ce que c'est un roman? Non, ce n'est pas un roman. Qu'est-ce que c'est? C'est un manuel.
  2. Est-ce que ce sont des chaises? Non, ce ne sont pas des chaises. Qu'est-ce que c'est? Ce sont des bancs.
  3. C'est Pascal? = Est-ce que c'est Pascal? = Est-ce Pascal?
  4. Où habite-t-elle? = Qù est-ce qu'elle habite?
  5. Ce sont des manteaux de Marie et de Monique.
  6. un roman = a novel, un manuel = a manual, a magnétophone = a tape recorder, un manteau = a coat, un banc = a bench, un document = a document, un disque = a disk, un peign = a comb, un lac = a lake, un pantalon = a pants, un timbre = a stamp, une radio = a radio, une robe = a dress, une moto = a motorbike, une chaise = a chair, une lettre = a letter, une cassette = a cassette, une brosse = a brush, une rivière = a river, une peinture = a paint, une photo = a photo, une chemise = a shirt, une enveloppe = an envelope, une pomme = an apple, une banane = a banana
  7. onze = 11, douze = 12, treize = 13, quatorze = 14, quinze = 15, seize = 16, dix-sept = 17, dix-huit = 18, dix-neuf = 19, vingt = 20

Leçon Sept

  1. Est-ce que vous êtes ingénieur? Oui, je suis ingénieur.
  2. Est-ce que monsieur Renou est ingénieur? Non, il n'est pas ingénieur. Il est technicien.
  3. Est-ce que vous êtes Chinoise? Oui, je suis Chinoise.
  4. Est-ce que madame Renou est Chinoise? Non, elle n'est pas Chinoise. Elle est Française.
  5. Où est la fleur? La fleur est sur la table. Elle est sur la table.
  6. un technicien/ une technicienne = a technician, un ingénieur/ une ingénieure = an engineer, un étudianté/une étudiante = a student, un ouvrier/une ouvriere = a worker, un médecin = a doctor, un pharmacien/une pharmacienne = pharmacist, un chanteur/une chantrice = a singer, un vendeur/une vendrice = a seller, un Chinois/une Chinoise = a Chinese, un Français/une Française = a French, un Espagnol/une Espagnole = a Spanish, un Américain/ une Américaine = an American, un Indien/une Indienne, un Japonais/une Japonaise
  7. la fleur = a flower, la table = a table, le tableau = a painting, le mur = a wall, le crayon = a pencil, le cahier = a notebook, le camion = a truck, le magasin = a ship, un aiparateur = a vacuum, la porte = a door, le ticket = a ticket, ma = my, la poche = a pocket, sur = on, sous = below, devant = in front, derrière = behind, dans = in
  8. vingt et un = 21, vingt-deux = 22, vingt-trois = 23, vingt-quatre = 24, vingt-cinq = 25, vingt-six = 26, vingt-sept = 27, vingt-huit = 28, vingt-neuf = 29, trente = 30
  9. J'aspirateur est derrière la chaise.
  10. J'habite à Beijing.
  11. être => je suis, tu es, il est, elle est, nous sommes, vous êtes, ils sont, elles sont

Leçon Huit

  1. Comment vous appelez-vous? Je m'appelle Li Long. Comment s'appelle-t-il. Il s'appelle Li Long.
  2. Quel âge avez-vous? J'ai dix-neuf ans.
  3. Est-ce que vous avez des frères et des sœurs? Oui, J'ai un frère, mai je n'ai pas de sœurs.
  4. Il va à l'école.
  5. un carte = a map, un soldat = a soldier, un officier = an officer, un dialogue = a dialogue, appeler = to call, comment = how, quel = what, un âge = an age, avoir = to have, un an = a year, un frère = a brother, a sœur = a sister, ma = my + masculine singular noun, mon = my + feminine singular noun, la famille = a family, le père = a father, la mère = a mother, travaille = work, une usine = a factory, dans une usine = in a factory, un instituteur/une institutrice = teacher, mais = but, une école = a school
  6. avoir => j'ai, tu as, il a, elle a, nous avons, vous avez, ils ont, elles, ont
  7. appeler => j'appelle, tu appeles, il appelle, elle appelle, nous appelons, vous appelez, ils appellent, elles appellent

Leçon Dix

  1. Se présenter

Bonjour. Je m'appelle Albert. Je suis Chinois. J'ai trente et un ans. Je suis étudiant. Ma famille habite à Péking. Mon père est professeur. Il travaille dans une université. Ma mère est comptable(accountant). Elle travaille dans une banque(bank). Je n'ai pas de frères et de sœurs. Je suis dans le département d'anglais. Je suis en troisième année. Nous avons beaucoup de professeurs d'anglais. J'apprends deux langues étrangères: l'anglais et le français. J,apprends le francais comme seconde langue étrangère. Le francais est difficile pour moi. Le masculin, le féminin, la conjugasion, tou ça, c'est très difficile. Je fais beaucoup d'exercices de français. Je viens à Canada pour apprendre le français. C'est une belle langue. Je fais mes études à l'université de la Colombie-Britannique. J'ai des cours du lundi au vendredi. Le soir, je fais mes devoirs dans ma chambre. Le samedi et le dimanche, je n'ai pas de cours. Je profite du temps libre pour visiter les monuments de Vancouver ou aller chez des amis canadien.
