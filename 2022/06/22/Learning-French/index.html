孙辉 《简明法语教程》

这里只列出句子用来背诵，其他的都在笔记上。

Leçon un

  1. Qui est-ce? C'est Pascal.
  2. Qù est-il? Il est à Nice.
  3. Qù est-elle? Elle est à Nice.

Leçon deux

  1. Est-ce que c'est Philippe? (Est-ce Philippe?) Oui, c'est Philippe.
  2. Que fait-il? El est chercheur.
  3. Est-ce que c'est Fanny? Oui, c'est Fanny.
  4. Que fait-elle? Elle est journaliste.
  5. Est-ce qu'il est à Nice? (Est-il à Nice?)
  6. Est-ce quèlle est à Nice? (Est-elle à Nice?)
  7. chercheur = researcher, acteur = actor, actrice = actress, facteur = postman, factrice = postwoman, journaliste = journalist, styliste = stylist

Leçon trois

  1. Est-ce que Thomas est avocat? Oui, il est avocat.
  2. Où habite-t-il? Il habite à Paris.
  3. Est-ce que Monique est économiste? Oui, elle est économiste.
  4. Où habite-t-elle? Elle habite à Rome.
  5. pilote = pilot, avocat = male lawyer, avocate = female lawyer, moniteur = male monitor, monitrice = female monitor, professeur = teacher, économiste = economist, cinéaste = filmmaker
  6. Où est-ce qu'elle habite? Elle habite à Nice.

Leçon quatre

  1. Est-ce que c'est votre classe? Oui, c'est notre classe.
  2. Qui est votre professeur? Monsieur Wang Ling est notre professeur.
  3. Qui est votre chef de classe? Wu Feng est notre chef de classe.
  4. chimiste = chemist, secrétaire = secretary, monsieur = sir, votre = your, classe = class, notre = our, chef de classe = class leader

Leçon cinq

  1. Qu'est-ce que c'est? C'est un stylo.
  2. Qu'est-ce que c'est? C'est une jupe.
  3. Est-ce que c'est le vélo de Charles? Oui, c'est le vélo de Charles.
  4. Est-ce que c'est la revue de Sophie? Oui, c'est la revue de Sophie.
  5. Quelle est sa couleur? Il est rouge.
  6. Quelle est sa couleur? Elle est noire.
  7. un stylo = a pen, un vélo = a bike, un parfum = a perfume, un livre = a book, un drapeau = a flag, un papier = a piece of paper, un verre = a glass, une jupe = a skirt, une guitare = guitar, une voiture = a car, une revue = a magazine, une veste = a jacket, a serviette = a tower
  8. rouge = red, vert = masculine green, verte = feminine green, noir = masculine black, noire = feminine black, bleu = masculine blue, bleue = feminine blue, jaune = yellow
