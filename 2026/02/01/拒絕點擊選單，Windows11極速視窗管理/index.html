拒絕點擊選單，Windows11極速視窗管理

Posted on 2026-02-01 In Computer Science
Symbols count in article: 1.1k Reading time ≈ 1 mins.

在現代的開發流程中，特別是當我們使用像 Antigravity 這種整合了 Code 和 Agent 的介面時，傳統的左右 1:1 分割螢幕已經不夠用了。為了讓程式碼閱讀更舒適，我個人的黃金比例是：左側 2/3 顯示主戰場，右側 1/3 疊放輔助工具。

很多人會去點擊視窗右上角的最大化按鈕來選擇佈局，但那個動作太慢、太費神。今天分享如何透過 PowerToys 與 Windows 11 原生隱藏功能，讓你滑鼠一「甩」就完成定位。

準備工作：下載 Microsoft PowerToys

雖然 Windows 11 有內建功能，但若要達成「最省力」的自定義佈局，我們需要微軟官方開發的開源工具 PowerToys。

  1. 前往下載： 到 Microsoft PowerToys 官方 GitHub 或直接在 Microsoft Store 搜尋並安裝。
  2. 啟動軟體： 安裝後打開 PowerToys 設定面板，確保 「FancyZones」 功能已開啟。

方案一：專業派 ── PowerToys FancyZones (最速吸附)

這是我最推薦的方法，因為它不需要精確瞄準，只要按住 Shift 就能讓視窗自動歸位。

1. 設定自定義 2:1 佈局

  • 進入 FancyZones 設定，點擊「啟動佈局編輯器」。
  • 選擇一個三欄佈局，按住滑鼠左鍵合併 (Merge) 左側兩個格子，形成一個 2/3 的大區域。
  • 關鍵細節： 關閉「區域周圍的空間」（Space around zones 切換為 Off），這樣視窗才會緊貼邊緣，沒有縫隙。

2. 滑鼠一「甩」就定位

這套動作不需要點擊任何選單： 1. 滑鼠按住 Antigravity 視窗的標題列。 2. 按下鍵盤 Shift 鍵，此時螢幕會出現你設定好的藍色影子。 3. 滑鼠隨意往左邊一甩，放開按鍵，視窗就會精準吸附到左側 2/3 區域。 4. 同樣的方法，將 VS Code 與 Chrome 甩到右側 1/3 區域，讓它們完美重疊。

方案二：直覺派 ── 螢幕頂部拖曳法 (免安裝)

如果你暫時不想安裝軟體，Windows 11 其實有一個隱藏的「頂部貼齊列」。

操作步驟：

  1. 按住視窗標題列，直接往螢幕正中央的最上方邊緣 (Top Center) 推過去。
  2. 此時上方會滑出一個佈局選單。
  3. 將滑鼠移入選單中「左 2/3」的格子並放開。

這比去瞄準視窗右上角的選單快得多，因為螢幕邊緣是「無限大」的目標，操作起來非常爽快。

為什麼這樣配置？

當我將 Antigravity 放在左側 2/3 時，寬度足以讓程式碼與 AI Agent 介面並排而不顯擁擠。

右側剩餘的 1/3 空間，我會同時把 VS Code 和 Chrome 定位在那裡。當我需要查看文檔時，用 Alt + Tab 切換即可。由於左右分區明確，大腦不需要在混亂的視窗堆裡尋找目標，能極大程度地保持心流 (Flow) 狀態。

如果你也覺得頻繁調整視窗大小很浪費時間，強烈建議試試這套「去選單化」的視窗管理術！
